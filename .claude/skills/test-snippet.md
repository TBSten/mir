# test-snippet

snippet の動作テスト (dry-run) を行う skill。

## 手順

1. 対象の snippet を確認する
2. テスト用の変数値を決定する (デフォルト値があればそれを使用)
3. 一時ディレクトリにインストールを実行する:
   ```bash
   npx mir install <name> --out-dir=/tmp/mir-test-<name> --key1=value1 ...
   ```
4. 生成されたファイルを確認する:
   - ファイル構造の確認
   - テンプレート変数が正しく展開されているか確認
   - hooks が正しく動作するか確認
5. 問題があれば修正を提案する
6. テスト用ディレクトリを削除する

## 注意事項

- 実際のプロジェクトディレクトリにはファイルを生成しない
- 変数のバリデーション (型チェック) も確認する
- `mir sync` で変数の過不足がないか事前に確認する
